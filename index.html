My Passion

In my 12th grade I finally felt like what I really wanna do all my life. A time line of my few Projects.

20th Nov 2014

Electric Bill Management System

This Project Was made in Cpp, It had all the necessary features, File based CLI software which could
be used my multiple users with different bill numbers, It will tell the user how much bill should be
paid based on the units consumed

13th March 2016

Calculator

This Project was a very small Calculator project made in JAVA, it had all arithmetic operation
capability. It was a CLI project but had better option and very guiding interface for the user despite
being without a GUI.

21th June 2018

Pass Protect

This was My first project That was GUI based, Made in Visual Cpp. It was made in order to have a
Secure offline system to safely store multiple passwords at a single place without any need to write
those passwords somewhere physically. It had a feature of multiple user accounts so many people can
use it on the same system.

3rd August 2018

Crypter

This project was made in python, it was a script that was capable of taking any sentence and then
encrypting it into something unreadable. This unreadable text could only be read by the person with
this script and the key assigned to the text. I made sure the script doesnâ€™t crash no matter what the
user tries to input.

12th February 2019

Light IOT

This was my first IOT project, made by using BOLT IOT kit, the iot device was programmed to send
data to a online dash board. The device will collect the data about how bright the place was through
out the day, and that data could be retrieved from the dash board in a CSV file for further data
mining.

Ongoing

Secure Vault

This is an remake of the Pass protect project made in C#, this has a better looking UI, supports Win XP,
Win 7, Win 8, Win 10. It has 3 security layers, Tested on 8 Systems and gets regular updates. Easy to
install with a MSI installer. Msi is signed with a self-signed certificate., Free to download and use.
